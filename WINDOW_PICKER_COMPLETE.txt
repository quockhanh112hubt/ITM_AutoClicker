TECHNICAL DETAILS
================================================================================

Windows API Used (via pywin32):
  - win32gui.EnumWindows() - List all windows
  - win32gui.GetWindowText() - Get window title
  - win32gui.GetClassName() - Get window class
  - win32gui.GetWindowRect() - Get window position
  - win32gui.IsWindowVisible() - Check if visible

Coordinate System:
  - Local coords: Within window (0-based, relative to window)
  - Global coords: Relative to screen top-left (0, 0)
  - Conversion: global = window_pos + local

PyQt6 Used:
  - WindowPickerDialog(QDialog)
  - WindowRegionSelector(QWidget) with FramelessWindowHint

Signals:
  - WindowPickerDialog.window_selected(Window)
  - WindowRegionSelector.region_selected(x1, y1, x2, y2)
