WHAT WAS BUILT
==============

An advanced Python application for automating mouse clicks with two modes:

1. POSITION-BASED CLICKS
   - Click at specific screen coordinates
   - Record positions with PAGE UP key
   - Exit with ESC key
   - Fast and reliable

2. IMAGE-BASED CLICKS
   - Find images on screen and click
   - Uses OpenCV template matching
   - Region selector overlay
   - Works with dynamic UIs

TECHNOLOGIES USED
=================

GUI Framework:
  - PyQt6 (Modern, Feature-rich)

Automation:
  - pyautogui (Mouse control)
  - pynput (Global keyboard listening)

Computer Vision:
  - OpenCV (Template matching)
  - Pillow (Image processing)

Data Management:
  - JSON (Script serialization)
  - Python Config (Settings management)

======================

Step 1: Create Virtual Environment

Step 2: Activate Virtual Environment
  Windows: .venv\Scripts\activate
  Mac/Linux: source .venv/bin/activate

Step 3: Install Dependencies

KEYBOARD HOTKEYS
================

PAGE UP  - Record position or confirm click location
ESC     - Exit recording mode
END     - Toggle start/stop execution

These work globally (even when app is not focused)

TROUBLESHOOTING
===============

Issue: "ModuleNotFoundError"
Solution: pip install -r requirements.txt

Issue: Clicks not working
Solution: Run as Administrator

Issue: Image not found
Solution: 
  - Increase click delay
  - Check image quality
  - Lower confidence threshold

Issue: Hotkeys not responding
Solution:
  - Restart application
  - Check for key conflicts
  - Run as Administrator
