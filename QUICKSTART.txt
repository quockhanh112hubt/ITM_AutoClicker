ITM AutoClicker - Ung Dung Tu Dong Click Chuot
================================================

QUICK START (Co tuc nhat):
==========================

1. Bat dau:
   - Open Command Prompt or PowerShell
   - Navigate to: E:\GitHub\ITM_AutoClicker
   - Type: python main.py

2. Neu loi (ImportError):
   - Type: python -m venv .venv
   - Type: .venv\Scripts\activate
   - Type: pip install -r requirements.txt
   - Type: python main.py

FEATURES:
=========
✓ Click theo vi tri (toa do x, y)
✓ Click theo hinh anh (template matching)
✓ Phim tac: PAGE UP, ESC, END
✓ Luu/Tai kich ban (JSON format)
✓ Delay co the tuy chinh (ms)
✓ GUI dung PyQt6

HOW TO USE:
===========

1. Mo ung dung:
   python main.py

2. Tao kich ban:
   - Click "Add Action"
   - Chon "Position Based Click" hoac "Image Based Click"

3. Position-Based Click:
   - Di chuot den vi tri muon click
   - Nhan PAGE UP de ghi
   - Nhan ESC khi xong

4. Image-Based Click:
   - Keo chuot chon vung anh
   - Thaa chuot
   - Di chuot den vi tri click
   - Nhan PAGE UP
   - Chon Yes hoac No

5. Chay kich ban:
   - Click "Start" hoac nhan END
   - Click "Stop" hoac nhan END de dung

PHIM TAC (HOTKEYS):
===================
PAGE UP   -> Ghi vi tri / Xac nhan click
ESC       -> Thoat mode ghi
END       -> Bat dau / Dung

SETTINGS:
=========
Tab "Settings" de cai dat:
- Click Delay: Thoi gian giua cac click (ms)
- Default: 100ms
