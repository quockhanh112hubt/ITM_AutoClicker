ARCHITECTURE:

    └─ Creates ImageRecordingManager
       ├─ Manages keyboard listener
       ├─ Coordinates dialogs

DATA FLOW:

    → Region selector
    → User drags region
    → ImageConfirmationDialog shows preview
    → User clicks OK/Cancel
    → ClickPositionDialog appears
    → User presses PAGE UP to record position
    → Ask "Continue recording?"
    → If Yes: Loop back to region selector
    → If No: Call on_complete(recorded_images)
    → MainWindow adds images to script

HOW TO USE THE NEW WORKFLOW
================================================================================

1. Run the application:
   python main.py

2. Click "Add Action" button

3. Select "Image Based Click"

4. Drag to select an image region on screen

5. Review image in preview dialog
   - Click OK to confirm
   - Click Cancel to reselect

6. Move mouse to desired click location

7. Press PAGE UP to record position

8. Dialog shows position, asks to continue
   - Click Yes to record more images
   - Click No to finish
   - Press ESC to exit

9. All recorded images appear in script table

10. Execute script to test clicks work
